⚠️ ANTI-MANIPULATION

Anti-Trap Protocols

Market Maker selalu mencari Exit Liquidity. Kenali jebakan mereka sebelum jadi korban.

JebakanModus OperandiCounter-Tactic
Order Book Spoofing
Dinding ilusi di Bid
Pasang ratusan ribu lot palsu di Bid untuk pancing ritel HAKA. Detik ritel masuk, antrean dicabut, saham dibuang.
→ Tape Reading
Fokus pada Running Trade (Match). Jika antrean raksasa hilang tanpa ada volume eksekusi → Spoofing. Jangan entry.
Nominee Illusion
Paus berbaju ritel
Institusi memecah akumulasi ke puluhan akun broker ritel kelas bawah. Screener standar anggap sebagai noise ritel.
→ ATV Analysis
Hitung Average Transaction Value = Total Value ÷ Frekuensi. Nilai besar + frekuensi rendah = HNWI menyamar.
Fake Breakout (Bull Trap)
Chart breakout palsu
Harga ditarik lewat resisten teknikal. Semua indikator ritel hijau. Ritel HAKA. Bandar distribusi di puncak.
→ VPA Exhaustion
Jika Harga NAIK + Volume TURUN = anomali fatal. Market Maker menarik gas. Take Profit atau jangan entry.
Shakeout Markdown
Penurunan fiktif
Harga dibanting merah berturut-turut untuk paksa weak hands cut loss massal. Ritel panik.
→ VPA Absorption
Jika Harga TURUN + Volume KERING/MENYUSUT = Healthy Retracement. Institusi menadah, bukan buang. HOLD.
Dark Pool Discount Illusion
Pasar Negosiasi
Crossing raksasa di Pasar Nego dengan harga diskon besar. Ritel salah artikan sebagai akumulasi murah.
→ Price Delta Check
Diskon crossing = bailout/repo/tukar guling afiliasi. HANYA valid jika Premium (di atas harga reguler) = sinyal M&A.
Sell-on-News Slaughter
Berita bagus = exit bandar
Corporate action resmi rilis. Media heboh. Ritel FOMO borong di pucuk. Bandar buang ke ritel.
→ Sentiment vs Flow
Jika hype sosial memuncak TAPI institusi Net Sell di Broker Summary → berita itu alat exit liquidity. Tinggalkan.
Painting the Tape
Volume palsu / Wash Trading
Transaksi jual-beli ke diri sendiri untuk ciptakan volume palsu. Masuk Top Active screener ritel.
→ Gross vs Net
Gross Value besar tapi Net Value (Beli-Jual) mendekati nol = Wash Trading. Flag sebagai Toxic Liquidity, abaikan.
Index Rebalancing Front-Run
Pembantaian hari MSCI
Saham masuk MSCI/FTSE. Ritel tahan barang menunggu asing beli di Hari-H. Smart Money sudah akumulasi jauh sebelumnya dan jual ke asing itu.
→ Event Timeline Arbitrage
Buy the Rumor (akumulasi saat sepi), Sell BEFORE the Fact. Likuidasi sehari sebelum atau tepat di Hari-H rebalancing.
Warrant Sweetener Trap
IPO / RI dengan waran pemanis
Waran gratis 1:1. Institusi jual waran di pucuk hari-1 → modal balik → buang saham induk habis-habisan. Ritel yang hold induk kena ARB.
→ Mother Share Liquidation
Jangan hold saham induk IPO/RI berstruktur waran besar. Jual saham induk di hari-1 bersamaan dengan listing waran.
Dilution Scheme (RI Jumbo)
Rights Issue gali-tutup lubang
RI jumbo berdalih ekspansi, nyatanya bayar utang afiliasi. Ritel yang tidak tebus terdilusi. Yang tebus terus suntik modal.
→ Standby Buyer Validation
HANYA valid jika Konglo berstatus Standby Buyer di harga Premium. Tanpa itu = Toxic RI, skip.
