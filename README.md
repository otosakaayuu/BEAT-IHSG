## 🛠️ Deploy ke GitHub Pages

### 1. Buat Repository Baru
```bash
git init
git add .
git commit -m "Initial: BEAT IT Strategy Checklist"
git branch -M main
git remote add origin https://github.com/USERNAME/beat-it.git
git push -u origin main
```

### 2. Aktifkan GitHub Pages
1. Buka **Settings** → **Pages**
2. Source: pilih **GitHub Actions**
3. Push ke `main` branch → auto deploy!

### 3. Akses
```
https://USERNAME.github.io/beat-it/
```
